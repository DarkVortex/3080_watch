# 3080_watch

In order to properly install this, you will need:

-bs4

-requests

-smtplib

-time

-os

-json

-pathlib

-difflib

You will also need to create a "webhooks.json" file in the same directory as start.py, of the following format:

{

    "WEBHOOKS":[
        WEBHOOK_1,

        WEBHOOK_2,

        etc.

    ]
}
